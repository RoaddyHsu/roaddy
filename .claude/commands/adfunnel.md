## 二、各階段策略

### 2.1 TOFU - 認知層（Top of Funnel）

**目標**：觸及新受眾、建立品牌認知

| 項目 | 設定 |
|------|------|
| 受眾 | 興趣/類似受眾 |
| 廣告目標 | 品牌認知/觸及 |
| 內容類型 | 品牌影片/形象圖 |
| 預算佔比 | 30% |
| KPI | CPM、觸及人數 |

**素材建議**
- 品牌形象影片（15-30秒）
- 產品魅力呈現
- 品牌故事

**CTA**：了解更多

---

### 2.2 MOFU - 考慮層（Middle of Funnel）

**目標**：深化興趣、建立信任

| 項目 | 設定 |
|------|------|
| 受眾 | 網站訪客/互動受眾 |
| 廣告目標 | 流量/互動 |
| 內容類型 | 產品介紹/社會證明 |
| 預算佔比 | 40% |
| KPI | CTR、互動率 |

**素材建議**
- 產品詳細介紹
- 顧客見證
- 使用教學

**CTA**：看看評價 / 了解產品

---

### 2.3 BOFU - 轉換層（Bottom of Funnel）

**目標**：促進轉換、完成購買

| 項目 | 設定 |
|------|------|
| 受眾 | 加購物車/高意圖 |
| 廣告目標 | 轉換/目錄銷售 |
| 內容類型 | 促銷優惠/限時活動 |
| 預算佔比 | 30% |
| KPI | CPA、ROAS |

**素材建議**
- 限時優惠
- 最後提醒
- 專屬折扣碼

**CTA**：立即購買 / 馬上訂購
