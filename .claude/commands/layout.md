## 三、格線系統

### 格線設定
| 項目 | 設定 | 說明 |
|------|------|------|
| 欄數 | 12欄 | 靈活分配 |
| 欄寬 | px | 依版面 |
| 間距 | px | 欄間距離 |
| 邊距 | px | 版面邊緣 |

### 格線視覺化
```
┌──────────────────────────────────────────┐
│←邊距→│ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │←邊距→│
│      │   │   │   │   │   │   │      │
│      ├───┴───┴───┴───┴───┴───┤      │
│      │      內容區域          │      │
│      │                        │      │
│      │    ┌────┐ ┌────────┐  │      │
│      │    │4欄 │ │  8欄   │  │      │
│      │    └────┘ └────────┘  │      │
│      │                        │      │
└──────────────────────────────────────────┘
```

### 常用欄位組合
| 組合 | 比例 | 適用 |
|------|------|------|
| 12 | 全版 | 大圖、標題 |
| 6+6 | 1:1 | 雙欄對比 |
| 4+8 | 1:2 | 側邊+主內容 |
| 3+6+3 | 1:2:1 | 居中內容 |
| 3+3+3+3 | 1:1:1:1 | 四格均分 |

## 四、視覺層級

### 層級設定
| 層級 | 元素 | 視覺處理 | 佔比 |
|------|------|----------|------|
| 1 | 主標題 | 最大、最粗 | 10% |
| 2 | 副標題 | 次大 | 10% |
| 3 | 重點資訊 | 中等強調 | 20% |
| 4 | 說明文 | 標準大小 | 40% |
| 5 | 次要資訊 | 較小 | 20% |

### 層級對比方式
| 方式 | 說明 | 效果 |
|------|------|------|
| 大小對比 | 字級差異 | 最直接 |
| 粗細對比 | 字重差異 | 細膩 |
| 色彩對比 | 深淺/彩度 | 情感表達 |
| 位置對比 | 上/下、左/右 | 結構感 |
| 空間對比 | 留白多寡 | 重要性 |

### 層級檢核
| 檢核 | 說明 | 狀態 |
|------|------|------|
| 瞇眼測試 | 瞇眼時能看出主次 | ⬜ |
| 3秒測試 | 3秒內抓到重點 | ⬜ |
| 距離測試 | 遠看仍有層次 | ⬜ |

## 五、留白設計

### 留白類型
| 類型 | 說明 | 作用 |
|------|------|------|
| 外部留白 | 版面邊緣空間 | 框架感 |
| 內部留白 | 元素內padding | 呼吸感 |
| 元素間距 | 物件之間空間 | 群組區隔 |
| 行距 | 文字行間距 | 閱讀舒適 |

### 留白比例建議
| 版面類型 | 留白比例 | 說明 |
|----------|----------|------|
| 高端質感 | 40-50% | 奢侈品風格 |
| 平衡舒適 | 30-40% | 一般商業 |
| 資訊密集 | 20-30% | 促銷DM |

### 留白應用
| 場景 | 建議 |
|------|------|
| 標題周圍 | 至少1倍標題高度 |
| 段落間 | 1.5倍行距 |
| 區塊間 | 2-3倍基礎間距 |
| 版面邊緣 | 5-10%版面寬度 |

## 六、視覺流向

### 閱讀模式
| 模式 | 適用 | 設計建議 |
|------|------|----------|
| Z型 | 網頁、廣告 | 左上→右上→左下→右下 |
| F型 | 文字為主 | 上方橫讀、左側直讀 |
| 環形 | 海報 | 從中心向外發散 |
| 直線 | 長頁面 | 由上而下 |

### 引導技巧
| 技巧 | 說明 | 應用 |
|------|------|------|
| 箭頭指向 | 明確指引 | 按鈕引導 |
| 視線方向 | 人物看向 | 人物照片 |
| 對比吸引 | 突出元素 | 重點強調 |
| 動線安排 | 順序配置 | 步驟說明 |

### 視覺流向檢核
```
起點（最大/最亮）
    ↓
次要重點
    ↓
說明內容
    ↓
行動呼籲（CTA）
    ↓
終點（品牌/聯絡資訊）
```

## 七、文字排版

### 中文排版要點
| 要點 | 說明 | 建議值 |
|------|------|--------|
| 行長 | 每行字數 | 25-35字 |
| 行距 | 行間距離 | 1.5-1.8倍 |
| 字距 | 字元間距 | 0-5% |
| 段距 | 段落間距 | 1行高度 |

### 標題設計
| 類型 | 字級參考 | 字重 |
|------|----------|------|
| 大標題 | 32-48px | Bold |
| 中標題 | 24-32px | SemiBold |
| 小標題 | 18-24px | Medium |
| 內文 | 14-16px | Regular |

### 文字對齊
| 對齊方式 | 適用場景 | 避免 |
|----------|----------|------|
| 左對齊 | 內文、列表 | ✓ |
| 置中對齊 | 標題、短文 | 長段落 |
| 右對齊 | 數字、日期 | 中文內文 |
| 分散對齊 | 報紙風格 | 一般設計 |
