## 三、基礎調整

### 裁切與構圖
| 構圖法則 | 說明 | 適用 |
|----------|------|------|
| 三分法 | 九宮格交叉點放主體 | 一般照片 |
| 中央構圖 | 主體置中 | 產品照 |
| 黃金比例 | 1.618比例分割 | 精緻構圖 |
| 框架構圖 | 利用環境框住主體 | 情境照 |

### 色彩校正

|----------|------|--------|

| 陰影 | 暗部細節 | +10~+30 |
| 白平衡 | 色溫校正 | 依光源 |
| 飽和度 | 色彩鮮豔度 | +5~+15 |

## 七、輸出規範

### 檔案格式選擇
| 格式 | 特性 | 適用 |
|------|------|------|
| JPG | 有損壓縮、小檔 | 照片、網頁 |
| PNG | 支援透明、無損 | 去背圖、圖示 |
| GIF | 動畫、有限色 | 簡單動畫 |
| WEBP | 高壓縮比 | 網頁優化 |
| TIFF | 無損、大檔 | 印刷 |
| PSD | 可編輯原始檔 | 存檔用 |

### 輸出品質設定
| 用途 | 品質 | 檔案大小 |
|------|------|----------|
| 網頁用 | 60-80% | <500KB |
| 社群用 | 80-90% | <1MB |
| 印刷用 | 100% | 不限 |

### 色彩模式
| 模式 | 適用 | 注意 |
|------|------|------|
| sRGB | 網頁、螢幕 | 標準色域 |
| CMYK | 印刷 | 色彩會變 |
| Adobe RGB | 專業攝影 | 寬色域 |

## 九、常見問題處理

### 問題與解決
| 問題 | 原因 | 解決方案 |
|------|------|----------|
| 圖片模糊 | 解析度不足 | 使用AI放大/重拍 |
| 色差 | 色彩模式錯誤 | 轉換正確模式 |
| 檔案太大 | 未壓縮 | 輸出時壓縮 |
| 去背毛邊 | 選取不精準 | 使用遮色片修飾 |
| 色彩不鮮豔 | 彩度不足 | 適度提升飽和 |
| 曝光過度 | 拍攝或調整問題 | 壓暗高光 |

### 品質檢查
| 檢查項目 | 標準 |
|----------|------|
| 解析度 | 符合用途要求 |
| 色彩 | 準確、一致 |
| 邊緣 | 乾淨無毛邊 |
| 瑕疵 | 已清除 |
| 格式 | 正確 |
