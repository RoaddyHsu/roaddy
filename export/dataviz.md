## 四、圖表設計規範

### 顏色使用
| 用途 | 色碼 | 說明 |
|------|------|------|
| 主要數據 | 品牌色 | 核心指標 |
| 次要數據 | 輔助色 | 對比數據 |
| 正向 | #22C55E | 成長、達標 |
| 負向 | #EF4444 | 下降、未達 |
| 中性 | #9CA3AF | 參考線 |

### 長條圖規範
```
┌─ 標題清楚說明圖表內容 ─┐
│                        │
│ 類別A ████████ 80     │ ← 數值標籤
│ 類別B ██████ 60       │
│ 類別C ████ 40         │
│ 類別D ██ 20           │
│        ↑               │
│     從0開始            │
└────────────────────────┘
  ↓ 資料來源與日期
```

### 折線圖規範
```
┌─ 標題 ─────────────────┐
│ $                       │
│ 100k ─────────╮        │ ← Y軸刻度
│               │        │
│  50k ────╮    │        │
│          │    │        │
│   0k ────┴────┴────    │
│      1月  2月  3月     │
│      ── 系列A  -- 系列B│ ← 圖例
└────────────────────────┘
```

### 圓餅圖規範
- 類別不超過5-7個
- 從12點鐘方向順時針
- 最大佔比放第一
- 小於5%合併為「其他」

## 六、避免陷阱

### 常見錯誤
| 錯誤 | 問題 | 正確做法 |
|------|------|----------|
| Y軸不從0開始 | 誇大差異 | 從0開始或說明 |
| 3D效果 | 扭曲比例 | 使用2D圖表 |
| 太多顏色 | 難以區分 | 限制5色內 |
| 太多類別 | 無法閱讀 | 分群或篩選 |
| 無資料來源 | 缺乏信任 | 註明來源 |

### 檢核清單
| 項目 | 檢核 |
|------|------|

| ⬜ 軸線標示清楚 | |
| ⬜ 無視覺誤導 | |
| ⬜ 資料來源註明 | |
