## 四、公式庫

### 常用公式

**加總計算**
```excel
=SUMIF(範圍, 條件, 加總範圍)
=SUMIFS(加總範圍, 條件範圍1, 條件1, 條件範圍2, 條件2)
```

**查找計算**
```excel
=VLOOKUP(查找值, 範圍, 欄位, FALSE)
=INDEX(範圍, MATCH(查找值, 查找範圍, 0))
```

**條件計算**
```excel
=IF(條件, 真值, 假值)
=IFS(條件1, 值1, 條件2, 值2)
```

**日期計算**
```excel
=EOMONTH(TODAY(), 0)  '本月最後一天
=WEEKNUM(TODAY())     '本週週次
```
