| 對象 | 時機 | 內容 | 管道 |

| 加盟主 | 2小時內 | 說明與指導 | 電話 |

### 對外溝通
| 對象 | 原則 | 口徑 |
|------|------|------|
| 消費者 | 誠懇、負責 | 統一聲明 |
| 媒體 | 單一窗口 | 正式新聞稿 |
| 主管機關 | 配合、透明 | 據實說明 |
| 合作夥伴 | 主動告知 | 正式通知 |

### 聲明模板
```
【關於OOO事件聲明】

針對近日發生的OOO事件，弘爺漢堡鄭重聲明如下：

一、事件說明：
[簡述事件經過]

二、處理措施：
1. [立即採取的行動]
2. [後續改善措施]
3. [對受影響者的處理]

三、品牌承諾：
弘爺漢堡一向以[核心價值]為最高準則，
對於本次事件深表歉意，將持續精進，
確保提供消費者最優質的餐飲體驗。

如有任何疑問，歡迎洽詢客服專線：XXXX-XXXX

弘爺漢堡 敬上
[日期]
```

## 六、處理流程

### 黃金時間
| 時間 | 行動 | 負責人 |
|------|------|--------|
| 0-15分鐘 | 確認事實、初步應對 | 現場 |
| 15-30分鐘 | 通報層級、啟動預案 | 主管 |
| 30-60分鐘 | 組成小組、統一口徑 | 總部 |
| 1-2小時 | 對外聲明（如需要） | 發言人 |
| 24小時內 | 完整說明、後續措施 | 小組 |

### 處理步驟
```
1. 確認 → 2. 通報 → 3. 應對 → 4. 溝通 → 5. 恢復 → 6. 檢討
```
