## 四、考核流程

### 流程步驟
```
目標設定 → 期中回饋 → 自評 → 主管評核 → 審核確認 → 面談回饋 → 結果應用
```

### 各階段說明
| 階段 | 時間 | 負責人 | 產出 |
|------|------|--------|------|
| 目標設定 | 期初 | 主管+員工 | 目標卡 |
| 期中回饋 | 期中 | 主管 | 回饋記錄 |
| 自評 | 考核期 | 員工 | 自評表 |
| 主管評核 | 考核期 | 主管 | 評核表 |
| 審核確認 | 考核後 | HR/高主管 | 確認表 |
| 面談回饋 | 考核後 | 主管 | 面談記錄 |

---

## 五、績效面談

### 面談準備
| 項目 | 準備內容 |
|------|----------|
| 主管準備 | 評核結果、具體事例、發展建議 |
| 員工準備 | 自評結果、成果說明、問題反映 |

### 面談流程
| 步驟 | 時間 | 內容 |
|------|------|------|
| 開場 | 5分 | 說明目的、建立氛圍 |
| 回顧 | 10分 | 工作表現回顧 |
| 評核 | 10分 | 說明評核結果 |
| 回饋 | 10分 | 優點與改善 |
| 發展 | 10分 | 未來目標與發展 |
| 結尾 | 5分 | 確認、鼓勵 |

### 面談技巧
| 技巧 | 說明 |
|------|------|
| SBI法 | 情境→行為→影響 |
| 三明治法 | 正面→改善→正面 |
| GROW法 | 目標→現況→選項→意願 |

## 八、績效改善

### PIP (績效改善計畫)
| 項目 | 內容 |
|------|------|
| 適用對象 | 績效C或D級 |
| 期間 | 3個月 |
| 目標 | 提升至B級 |

### PIP流程
| 步驟 | 內容 | 負責人 |
|------|------|--------|
| 問題診斷 | 分析績效不佳原因 | 主管 |
| 計畫擬定 | 設定改善目標與行動 | 主管+員工 |
| 執行追蹤 | 每週檢視進度 | 主管 |
| 結果評估 | 評估是否改善 | 主管+HR |
