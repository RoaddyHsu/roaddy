## 二、互動貼文設計

### 2.1 問答式互動

**貼文內容**
```

🤔 [問題]

A. 選項一
B. 選項二
C. 選項三

留言告訴我們你的答案！👇

#弘爺漢堡
```

---

### 2.2 投票式互動（IG 限動）

**設計**
```
問題：

選項 A：
選項 B：
```

---

### 2.3 填空式互動

**貼文內容**
```

今天早餐我最想吃 _______ 🍔

留言完成句子，我們選 3 位送小禮物！🎁

#弘爺漢堡
```

---

### 2.4 Tag 好友式

**貼文內容**
```

你身邊有沒有 [特定描述] 的朋友？

Tag 他們來看！👇
標記最多的前 3 名，送 [獎品]！

#弘爺漢堡
```

---

### 2.5 抽獎互動式

**貼文內容**
```
🎁 [活動名稱] 抽獎活動 🎁

📌 參加方式：
1️⃣ 追蹤 @弘爺漢堡
2️⃣ 按讚這則貼文
3️⃣ Tag 2 位好友
4️⃣ 留言「[指定內容]」

🎁 獎品：
🏆 [獎品內容]

⏰ 活動期間：
📅 公布日期：

#弘爺漢堡 #抽獎
```
