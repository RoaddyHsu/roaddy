## 三、LTV 計算

### 基礎數據
| 指標 | 數值 | 計算方式 |
|------|------|----------|
| 平均客單價 (AOV) | $ | 總營收/訂單數 |
| 購買頻率 (F) | 次/年 | 年訂單數/客戶數 |
| 毛利率 | % | (營收-成本)/營收 |
| 客戶壽命 (L) | 年 | 1/年流失率 |
| 年流失率 | % | |

### LTV 計算
```
簡單 LTV = AOV × F × L
         = $ × × = $

毛利 LTV = AOV × F × L × 毛利率
         = $ × × × % = $
```

## 五、回本期分析

### CAC 回收期計算
```
回本期 = CAC / (月營收 × 毛利率)
       = $ / ($ × %) = 個月
```
